## Communication Plan

| Communication Area | Approach |
|--------------------|----------|
| Day-to-day Communication | Telegram or Whatsapp will be used for quick discussions, reminders, and immediate updates with each other |
| Scheduled Discussions | Weekly online meetings via Google Meet will be scheduled for  reviewing the progress and plan next tasks |
| File Sharing | Google Drive will be used for sharing the drafts, diagrams, and other reference materials |
| Task Allocation | Tasks will be divided at the start of each week and tracked using a shared checklist |
| Decision Approval | Technical and design decisions confirmed during meetings and documented |
| Submission Coordination | Final reviews will be conducted between the team members prior to the submission to ensure consistency |
